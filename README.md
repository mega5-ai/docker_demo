# docker_demo
This project is a simple Dog Image Classification Web Application built using FastAPI and a pretrained MobileNetV2 model from TensorFlow. The application allows users to upload an image through a web interface, and the model analyzes the image to predict whether it contains a dog or not. The prediction result along with the confidence score is displayed to the user.

The system includes a lightweight frontend built with HTML for image upload and result display, while the backend handles image preprocessing, model inference, and API responses. To ensure easy deployment and consistent execution across different environments, the entire application is containerized using Docker.

This project demonstrates how deep learning models can be integrated with web frameworks and deployed using containerization technologies to create simple and portable AI-powered applications. 🚀
